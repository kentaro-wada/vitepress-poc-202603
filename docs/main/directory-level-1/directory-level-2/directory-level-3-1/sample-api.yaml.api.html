ユーザー管理API ​

  • OpenAPI Version: 3.1.1
  • API Version: 1.0.0

ユーザーの取得・作成・更新・削除を行うAPIです。

Servers ​

  • URL: https://api.example.com/v1
    • Description: 本番環境
  • URL: https://api-stg.example.com/v1
    • Description: ステージング環境

Operations ​

ユーザー取得 ​

  • Method: GET
  • Path: /users/{userId}
  • Tags: Users

指定したIDのユーザー情報を取得します。

Responses ​

Status: 200 取得成功 ​
Content-Type: application/json ​

  • createdAt (required)

    string, format: date-time — 作成日時

  • email (required)

    string, format: email — メールアドレス

  • id (required)

    string — ユーザーID

  • name (required)

    string — 氏名

  • role (required)

    string, possible values: "admin", "member", "viewer" — ロール

Example:

json
{
  "id": "usr_abc123",
  "name": "山田 太郎",
  "email": "yamada@example.com",
  "role": "admin",
  "createdAt": "2024-01-15T09:00:00Z"
}
Status: 401 認証エラー ​
Content-Type: application/json ​

  • code (required)

    string — エラーコード

  • message (required)

    string — エラーメッセージ

Example:

json
{
  "code": "USER_NOT_FOUND",
  "message": "指定されたユーザーが存在しません"
}
Status: 404 ユーザーが見つからない ​
Content-Type: application/json ​

  • code (required)

    string — エラーコード

  • message (required)

    string — エラーメッセージ

Example:

json
{
  "code": "USER_NOT_FOUND",
  "message": "指定されたユーザーが存在しません"
}

ユーザー一覧取得 ​

  • Method: GET
  • Path: /users
  • Tags: Users

ユーザーの一覧を取得します。ページネーションに対応しています。

Responses ​

Status: 200 取得成功 ​
Content-Type: application/json ​

  • items (required)

    array

    Items:

    • createdAt (required)

      string, format: date-time — 作成日時

    • email (required)

      string, format: email — メールアドレス

    • id (required)

      string — ユーザーID

    • name (required)

      string — 氏名

    • role (required)

      string, possible values: "admin", "member", "viewer" — ロール

  • limit (required)

    integer — 1ページあたりの件数

  • page (required)

    integer — 現在のページ番号

  • total (required)

    integer — 総件数

Example:

json
{
  "items": [
    {
      "id": "usr_abc123",
      "name": "山田 太郎",
      "email": "yamada@example.com",
      "role": "admin",
      "createdAt": "2024-01-15T09:00:00Z"
    }
  ],
  "total": 42,
  "page": 1,
  "limit": 20
}
Status: 401 認証エラー ​
Content-Type: application/json ​

  • code (required)

    string — エラーコード

  • message (required)

    string — エラーメッセージ

Example:

json
{
  "code": "USER_NOT_FOUND",
  "message": "指定されたユーザーが存在しません"
}
